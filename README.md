#LinkIt jQuery Plugin#

**LinkIt** es un plugin de jQuery para establecer vinculos.

##Version##
0.2.0

##Usage##
    $('span').linkIt({
        href:'http://test.com',
        text: 'Click Me',
        target: '_blank' 
    });

##Vendors##
jQuery - [http://jquery.com](http://jquery.com)
